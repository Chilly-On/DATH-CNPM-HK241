### How to run the application

Download PostgreSQL, using SQL file from server/database Postgre.sql to set up the database with the following setup:

```
    user: "postgres",
    host: "localhost",
    database: "DATH",
    password: "123",
    port: 5432,
```

Run the following command:
```
    cd backend
    node server
```
